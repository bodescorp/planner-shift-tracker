# Changelog

## [2.0.0] - 2025-12-17

### Reestruturação Completa
- Reorganização da estrutura de pastas (src/css, src/js, src/assets)
- Renomeação de arquivos para padrões profissionais
- Atualização de todos os caminhos de referência

### Design Minimalista
- Remoção de emojis excessivos
- Foco em tipografia e hierarquia visual
- Interface limpa e profissional
- Cores sutis (vermelho para plantão, verde para folga)

### Menu Unificado
- Todas as opções movidas para menu hambúrguer
- Interface principal focada apenas nas atividades
- Melhor organização das funcionalidades

### Sistema Automático
- Detecção automática de plantão/folga baseado em ciclo
- Configuração única: usuário define apenas uma vez
- Sistema calcula automaticamente os próximos dias
- Verificação de mudança de dia a cada minuto

## [1.0.0] - 2025-12-16

### Lançamento Inicial
- Sistema de cronograma bi-semanal (Semana A/B)
- Rastreamento de atividades com checkboxes
- Sistema de hidratação com lembretes
- PWA completo (instalável e offline)
- Relatórios semanais
- Limpeza automática segunda-feira 00:30
- Responsivo (mobile/desktop)
